# 플레이봇 검키우기 통계

카카오톡 채팅을 기반으로, 통계를 분석해주는 간단한 스크립트 및 웹페이지입니다.

## GH page

https://banatic.github.io/sword-stat/view.html

> 데이터 셋이 충분치 않아 정확하진 않습니당

## 사용법

1. `git clone https://github.com/banatic/sword-stat.git`

2. PC 카카오톡에서 대화 내보내기 후, `sword-stat` 폴더 내에 txt들을 위치시킵니다.

3. `python analyze.py` 후, `view.html`을 웹브라우저로 열면 확인할 수 있습니다.
